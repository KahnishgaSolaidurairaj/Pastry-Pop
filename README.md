# Pastry-Pop

I created this game using Unity 3D. This is a short fun game where the player can move around the enviornment (w,a,s,d) and throw purple frisbees at the pastrys to collect points. To win the game you have to have hit all of the pastries avaliable in the game which is a total of 5. 

I utilized free unity assets for this game and they are listed below:
- Oak Trees from Ada_King’s “Free Trees”
- Flowers and srubs from Ultrasonic’s “Simple, Low-poly Decorative Plant Assets”
- Strawberry Dessert from KamiChan’s “3D Cutie Desserts”
- Mountains from GameWarming's "Autumn Mountain"
